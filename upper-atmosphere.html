Atmospheric layers

Layer particles · live

Each row toggles a layer's gradient shell + particle system. Particles jitter at the layer's thermal speed; flow regime shifts with density; storms (Ap ≥ 27) puff thermosphere & exosphere outward. Pulse fires on every refresh.

Field overlay

Vectorised fields rendered as instanced cones across each layer. Temperature shows ∇T (radial + dayside tilt, magnitude ∝ layer T). Radiation shows EUV + energetic-particle flux flowing inward (magnitude ∝ F10.7 + Ap, with cusp boost on the polar field lines).

Satellite drag analysis

Live drag pressure q = ½ρv² for each tracked object, using the current state's local ρ and circular orbital speed v = √(μ/(R+h)). Click any row to fly the camera there. Orbital elements from CelesTrak.
