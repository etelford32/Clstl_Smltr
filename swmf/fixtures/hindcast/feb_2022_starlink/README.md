# Fixture — Feb 2022 Starlink event

Synthetic stand-in for the BATS-R-US output of a hindcast replay over
2022-02-03 → 2022-02-05. The shape (quiet → Φ_PC ramp → HPI peak →
recovery) approximates the real storm but the values are plausible
fakes: this fixture exists so `hindcast_runner.py --dry-run` is
exercisable in CI before real L1 IMF data is sourced.

`mhd_output.json` is generated by `scripts/build_hindcast_fixture.py`.
Re-run that script if the schema in `hindcast_runner.py` changes.
