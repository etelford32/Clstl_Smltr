# Parker Physics App — etelford32

Interactive 3D/2D celestial simulations and real-time space weather:
- **threejs.html** — Solar System simulation (real-time planet positions)
- **earth.html** — Earth magnetosphere, weather, satellite data
- **sun.html** — Solar physics (chromosphere, corona, interior)
- **space-weather.html** — Space weather dashboard (NOAA/NASA live data)
- **star3d.html** — Three.js 3D stellar simulator
- **dashboard.html** — User dashboard
